# Express-TS Repo

This is a template repo for setting up Express.js in typescript. This repo mimics an average express repo with middlewares, Routes, and Controllers to form a fully functional web api.

Additionally fully-typed CORS handling is also incorporated.

## Installing Express(TS)

- Initiating Package.json: npm init -y
- Installing Express.js: npm i express
- Installing typescript: npm i -D typescript
- Installing typescript's types: npm i -D @types/express
- Installing a default tsconfig package: npm i -D @tsconfig/node22

**Note**: Use node version 22.18 or greater for this setup to work. Otherwise needs to compile ts to js and start server from dist folder.
